Project Overview
This is a high-performance React.js dashboard built to solve the "visibility" problem in AI-driven recruitment. While AI backends handle the screening, this interface provides recruiters with a real-time, data-driven view of candidate performance, match scores, and hiring stages.

I built this specifically to demonstrate my readiness for the Software Engineering Intern role, showing that I can bridge the gap between complex AI logic and user-centric frontend design.
